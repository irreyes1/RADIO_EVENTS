# IRREYES_HANDOVER_SIM

Simulador pedagógico de traspaso de celdas diseñado por Iñaki Rodríguez.

Este proyecto permite visualizar el comportamiento de eventos A1–A6 y B1–B2 en una red móvil LTE/NR.

## Estructura del proyecto
- `index.html`: interfaz principal
- `style.css`: estilos
- `script.js`: lógica base
- `data/event_table.csv`: tabla de referencia
- `README.md`: documentación del proyecto
